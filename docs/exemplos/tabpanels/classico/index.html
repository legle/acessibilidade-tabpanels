Guia 1

Este é um Painel de Guias Acessível, compatível com JAWS e NVDA, construído com CSS, JQuery e HTML5 usando Way-aria.

Adaptação feita por Leonardo Gleison Ferreira do modelo disponível na página: http://accessibility.athena-ict.com/aria/examples/tabpanel2.shtml.

Estas Abas podem ser clicadas com o mouse para mostrar as guias ocultas.

Download do exemplo (Arquivo tgz 78KB")

Guia 2

Quando Estas guias estão presentes em sites, incluindo os sites do Google, frequentemente, nós, pessoas com deficiência visual, não conseguimos ler usando a tecla tab ou as setas de navegação, motivo pelo qual usei o atributo tabindex=”0” em todos os elementos “li” da barra de guias, o que viabiliza que esses elementos recebam o foco de navegação.

A segunda função no arquivo actions.js permite que as setas possam ser utilizadas para navegar entre as guias, mudando automaticamente o painel ativo, ao mesmo tempo que força o leitor de telas a ler a guia selecionada setando o foco de navegação para o elemento.

Além disso ela permite que o usuário pressione “Enter” para mostrar o painel quando navegar usando a tecla tab.
