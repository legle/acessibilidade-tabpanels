Acessibilidade Web para Pessoas com Deficiência Visual

Esta simples p&aacut;gina &eacut; uma modesta contribuição para a comunidade de desenvolvedores web, que tem como objetivo contribuir para a evolução da acessibilidade na web para às pessoas com deficiência, oferecendo exemplos testados por elas de como usar recursos do HTML5, WAI-ARIA e JQuery, além de outras tecnologias que eventualmente surgirem ao longo do tempo.

A ideia é desenvolver e testar componentes, técnicas e recursos web e disponibilizar para que outros desenvolvedores saibam como fazer, por isso, se você desenvolveu algo que funciona, por favor, contribua com este projeto!
